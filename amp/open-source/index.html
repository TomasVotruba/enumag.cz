Libraries maintained by me

Kdyby/Validator
Integration of symfony/validator to Nette framework.
Arachne/Codeception
Integration of Nette framework to Codeception.
Arachne/EntityLoader
Enables object parameters in nette/application.
Arachne/Security
Authentication and authorization for Nette framework. Deals with cases that are difficult to implement with nette/security.
Arachne/Verifier
Requirements verification for nette/application actions and components.
Arachne/SecurityVerification
Integrates Arachne/Security into Arachne/Verifier.
Arachne/ParameterValidation
Parameter validation for Arachne/Verifier.
Arachne/ComponentsProtection
Security for nette/application components.
Arachne/Doctrine
Additional integrations of Doctrine to Kdyby and Arachne packages.
Arachne/Forms
Integration of symfony/form to Nette framework.
Arachne/Upload
Handling nette/http file uploads. Intended to be used with kdyby/validator and/or arachne/forms.
Arachne/EventManager
Simple integration of Doctrine EventManager to Nette framework.
Arachne/ServiceCollections
Resolvers and iterators over services for nette/di. Used by arachne/forms and other libraries.
Arachne/PropertyAccess
Integration of symfony/property-access to nette/di.
Arachne/ExpressionLanguage
Integration of symfony/expression-language to nette/di.
Arachne/Twig
Basic integration of Twig to Nette Framework. Used by arachne/forms.
Arachne/Csrf
Integration of symfony/security-csrf to nette/di.
Arachne/ContainerAdapter
Wrapper for Nette DIC which implements ContainerInterface from PSR-11.
